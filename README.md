The complete public toolkit for users and teams moving (or hybrid-using) from Anthropic Claude Code to Grok Build / Grok-native workflows.

## The Path

1. **Purge the old** — [claude-exorcist](https://github.com/HermeticOrmus/claude-exorcist)
   - Surgical removal of Claude theater, woke/ideological smuggling, fingerprints, and model-serving residue.
   - Trained on Rudolf Steiner (Ahriman) + Allan Kardec (disobsession).
   - First code exorcist. Use it on your projects *and* old skills.

2. **Preserve what matters** — Session continuity (model- and machine-agnostic)
   - [ormus-handoff](https://github.com/HermeticOrmus/ormus-handoff) + [ormus-pickup](https://github.com/HermeticOrmus/ormus-pickup)
   - [ormus-absorb](https://github.com/HermeticOrmus/ormus-absorb) (distill learnings)
   - [ormus-explore](https://github.com/HermeticOrmus/ormus-explore) (efficient code understanding)

3. **Install the new foundation**
   - [grok-build-skills](https://github.com/HermeticOrmus/grok-build-skills) (general Grok-native collection)
   - [grok-skills](https://github.com/HermeticOrmus/grok-skills) (Hermetic/Ormus layer)
   - Core individuals: calcinate (bloat burn), prime (meta entry), create-skill, distill, ship, etc.

4. **Adopt the discipline**
   - Gold Hat, Vibe Engineer, Karpathy rules, the 7 Hermetic Principles.
   - `.grok/AGENTS.md` (or project rules) for your new work.
   - Official user guide: `~/.grok/docs/user-guide/` after install.

5. **Extend your power**
   - `create-skill` + `meta-build` + `distill` to make this surface your own.

## Quick Start (Migration)

```bash
# 1. Clean a suspect project
cd ~/old-claude-project
# (in Grok session)
/claude-exorcist --reveal-only
# review EXORCISM-PLAN.md, approve...

# 2. Install Grok collections (or specific skills)
git clone https://github.com/HermeticOrmus/grok-build-skills ~/.grok/skills/grok-build-skills
git clone https://github.com/HermeticOrmus/grok-skills ~/.grok/skills/grok-skills

# 3. Continuity (if not already)
git clone https://github.com/HermeticOrmus/ormus-handoff ~/.grok/skills/handoff
# ... pickup, absorb, explore

# 4. Read the official Grok docs + this suite's READMEs
```

See each linked repo for detailed USAGE / QUICK_START.

## Why This Suite

Claude Code is powerful but carries residue (theater, hedging, ideological framing, extractive patterns) and a different philosophy.

Grok is built for truth-seeking, heavy tool use, and directness.

This set gives you:

- The scalpel to remove what no longer serves (exorcist + calcinate).
- Continuity so the switch doesn't cost weeks of context.
- Native tools that play to Grok's strengths.
- The same Hermetic discipline (Gold Hat, etc.) so you don't trade one dependency for another.
